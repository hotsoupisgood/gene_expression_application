# Running the Vue Project in Development

## Prerequisites
Ensure you have **Node.js** and **npm** (or **yarn**) installed.

### 1️⃣ Install Dependencies
If you haven't already installed the project dependencies, run:
``` sh
npm install

npm run dev

```
